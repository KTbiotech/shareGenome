# shareGenome
Tạo super genome  từ nhiều genome của cùng một loài
```shell
usage: shareGenome.py [-h] [-l LISTFILES] [-k KMERSIZE] [--method-B METHOD]
                      [-g NUCMERG] [-b NUCMERB] [-c NUCMERC] [-i NUCMERIDY]
                      [--overlap-size OVERLAP]

optional arguments:
  -h, --help            show this help message and exit
  -l LISTFILES, --list-files LISTFILES
  -k KMERSIZE, --kmer-size KMERSIZE
  --method-B METHOD
  -g NUCMERG
  -b NUCMERB
  -c NUCMERC
  -i NUCMERIDY
  --overlap-size OVERLAP


```
